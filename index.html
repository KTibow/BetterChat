BetterChat

The first, only, and best chat filter for Hypixel.

What is BetterChat?

Better Chat is a ChatTriggers mod to improve the chat experience on Hypixel Minecraft. It rewrites chat messages to make them consistent, and hides spam.

How does it work?

BetterChat uses Bayesian analysis. It uses a list of past messages, and looks at the words commonly used for spam. If a message contains mostly words that are used in spam, it hides it.
